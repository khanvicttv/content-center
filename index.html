SETTINGS
GITHUB SYNC — AUTO SYNC ACROSS ALL DEVICES
Your data syncs automatically to a private file in your GitHub repo every time you make a change. Open the app on any device and it loads the latest data instantly — no manual export/import needed.

Setup: github.com → Settings → Developer settings → Personal access tokens → Tokens (classic) → Generate new token → check repo scope → copy it.
ANTHROPIC API KEY — AI CAPTION GENERATION
Your key is stored only in your browser (localStorage) and never sent anywhere except directly to Anthropic's API when you generate captions.

Get a free key: Go to console.anthropic.com → API Keys → Create Key. You get $5 free credit — enough for thousands of caption generations.
